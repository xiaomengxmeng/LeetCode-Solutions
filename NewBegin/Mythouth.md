```cpp
vector<vector<int>> merge(vector<vector<int>>& intervals) {
    vector<vector<int>> res;
    if (intervals.empty()) return res;
    
    sort(intervals.begin(), intervals.end());  // 按起点排序
    res.push_back(intervals[0]);               // 初始化
    
    for (int i = 1; i < intervals.size(); i++) {
        if (res.back()[1] >= intervals[i][0]) {
            // 重叠：合并，更新终点
            res.back()[1] = max(res.back()[1], intervals[i][1]);
        } else {
            // 不重叠：添加新区间
            res.push_back(intervals[i]);
        }
    }
    return res;
}
```

- ✅ **res.back()**：获取结果数组最后一个区间

## 2026-04-21 | [Medium] Linked List Cycle II (142)
### 🤔 我的原始思路
相遇点不是环的入口
相遇点和头节点同时再走可以走到入口！

### 🔍 我的思考过程

**Step 1: 问题分析**
1. 核心要求：找到环的入口节点
2. 与 [141] 的区别：需要定位，不只是判断

**Step 2: 数学推导**
```
设：a = 头到入口，b = 入口到相遇点，c = 相遇点到入口

慢指针：a + b
快指针：a + b + nr（n圈）

2(a + b) = a + b + nr
a = nr - b = c

结论：a = c（头到入口 = 相遇点到入口）
```

**Step 3: 两阶段法**
- 阶段1：快慢指针找相遇点
- 阶段2：两个指针从头和相遇点同时出发，相遇即为入口

**Step 4: 代码实现**
```cpp
// Step 1: 找相遇点
while (fast && fast->next) {
    slow = slow->next;
    fast = fast->next->next;
    if (slow == fast) break;
}
if (!fast || !fast->next) return nullptr;

// Step 2: 找入口
ListNode *p1 = head, *p2 = slow;
while (p1 != p2) {
    p1 = p1->next;
    p2 = p2->next;
}
return p1;
```

| [142] | 找入口节点 | 相遇后二次出发 |

**新学到的知识点**：
1. **数学推导能力**：通过公式推导证明 a = c
2. **两阶段算法**：先找相遇点，再找入口
3. **空间优化**：O(1) 空间解决定位问题

偶数 因为返回第二个中间节点 slow 再走一步
循环条件 fast!=nullprt||fast->next!=nullptr

### 🔍 我的思考过程

**Step 1: 问题分析**
1. 核心要求：找链表中间节点，偶数返回第二个
2. 快慢指针：快指针走2步，慢指针走1步

**Step 2: 奇偶分析**
```
奇数 [1,2,3,4,5]：
slow: 1→2→3, fast: 1→3→5
fast->next=nullptr 时结束，slow=3 ✅

偶数 [1,2,3,4,5,6]：
slow: 1→2→3→4, fast: 1→3→5→nullptr
fast=nullptr 时结束，slow=4 ✅（第二个中间节点）
```

**Step 3: 循环条件修正**
- ❌ 原始：`fast != nullptr || fast->next != nullptr`
- ✅ 正确：`fast && fast->next`
- 原因：`||` 会导致逻辑错误，必须两个条件都满足才能继续

**Step 4: 代码实现**
```cpp
ListNode* middleNode(ListNode* head) {
    ListNode *slow = head, *fast = head;
    while (fast && fast->next) {
        slow = slow->next;
        fast = fast->next->next;
    }
    return slow;
}
```
