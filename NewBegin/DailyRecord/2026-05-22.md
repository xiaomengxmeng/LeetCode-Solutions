## [Easy] 二叉树的直径 (543)
### 🎯 核心思路
后序遍历求高度，在递归过程中更新全局最大直径（左子树高度 + 右子树高度）。直径 = 边数，不是节点数。

### 🔑 关键代码片段
```cpp
int ans = 0;
int dfs(TreeNode* root) {
    if (!root) return 0;
    int L = dfs(root->left);
    int R = dfs(root->right);
    ans = max(ans, L + R);    // 以当前节点为最高点的直径
    return max(L, R) + 1;     // 返回子树高度（边数）
}
int diameterOfBinaryTree(TreeNode* root) {
    dfs(root);
    return ans;
}
```

### 📚 学到的知识点
1. **高度伪装求直径**：函数名义上求高度，实际用全局变量收集直径信息
2. **直径公式**= 左子树高度（边数）+ 右子树高度（边数），不是 `+1`
3. **后序遍历框架**：先递归子树，再处理当前节点——与 110、236 同模式

### ⚠️ 易错点 & 反思
- ❌ 以为直径要 +1（节点数思维）→ ✅ 直径是边数，`L + R` 即可
- ❌ 只考虑穿过根节点的路径 → ✅ 每个节点都可能是"最高点"，用全局变量取 max
- 成员变量 `ans` 在多次调用时需重置，LeetCode 场景无问题

## [Medium] Longest Univalue Path (687)
### 🎯 核心思路
543 模板 + 值不同就清零。后序遍历求高度，孩子值与当前节点不同时贡献清零，相同才累加。

### 🔑 关键代码片段
```cpp
int ans = 0;
int dfs(TreeNode* root) {
    if (!root) return 0;
    int L = dfs(root->left);
    int R = dfs(root->right);
    if (root->left && root->left->val != root->val) L = 0;
    if (root->right && root->right->val != root->val) R = 0;
    ans = max(ans, L + R);
    return max(L, R) + 1;
}
```

### 📚 学到的知识点
1. **543 模板的泛化**：同值路径 = 直径模板 + 值判断，仅多 2 行代码
2. **短路求值的妙用**：`root->left && root->left->val != root->val` 一句完成空指针检查和值比较
3. **独立清零**：左右子树互不影响，各自判断后独立决定是否清零

### ⚠️ 易错点 & 反思
- ❌ 容易忘记检查 `root->left` / `root->right` 是否存在，导致空指针访问
- ✅ `root->left && root->left->val != root->val` 短路求值保证安全
- 与 543 同模，上手极快

- [543] Diameter of Binary Tree — 基础模板，无值限制
- [250] Count Univalue Subtrees — 同值子树的计数变体
