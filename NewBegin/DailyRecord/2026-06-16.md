## [Medium] Target Sum (494)
### 🎯 核心思路
- 数学化简：P - N = target, P + N = sum → P = (target + sum) / 2
- 转化为：找出和为 P 的子集个数（0-1背包计数）
- 边界：abs(target) > sum 或 (target + sum) % 2 ≠ 0 → 0

### 🔑 关键代码片段
```cpp
int sum = accumulate(nums.begin(), nums.end(), 0);
if (abs(target) > sum || (target + sum) % 2 != 0) return 0;
int P = (target + sum) / 2;
if (P < 0) return 0;

vector<int> dp(P + 1, 0);
dp[0] = 1;
for (int num : nums)
    for (int j = P; j >= num; j--)
        dp[j] += dp[j - num];
return dp[P];
```

### 📚 学到的知识点
- 0-1背包从 bool（存在性）到 int（计数）的自然延伸
- 一维倒序 DP 的标准写法
- 问题转化的数学技巧：符号分配 → 子集和

### ⚠️ 易错点 & 反思
- 一维 dp 大小是 P+1 不是 n+1
- 漏掉 abs(target) > sum 检查会导致数组越界（P可能为负）
- 一维倒序遍历是 0-1 背包的固定要求
