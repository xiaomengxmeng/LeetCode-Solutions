## [Medium] 单词拆分 (139)

### 🎯 核心思路
布尔型一维 DP：`dp[i]` 表示 s[0..i)（前 i 个字符）能否由字典中的单词拼成。转移时枚举所有分割点 j，若 `dp[j]==true` 且 `s[j..i)` 在字典中，则 `dp[i]=true`。

### 🔑 关键代码片段
```cpp
bool wordBreak(string s, vector<string>& wordDict) {
    unordered_set<string> set;
    int maxSize = 0;
    for (string word : wordDict) {
        set.insert(word);
        maxSize = max(maxSize, (int)word.size());
    }
    vector<bool> dp(s.size() + 1, false);
    dp[0] = true;
    for (int i = 1; i <= s.size(); i++) {
        for (int j = 0; j < i; j++) {
            if (i - j > maxSize) continue;          // 剪枝
            if (dp[j] && set.find(s.substr(j, i - j)) != set.end()) {
                dp[i] = true;
                break;  // 找到一个即可
            }
        }
    }
    return dp[s.size()];
}
```

### 📚 学到的知识点
- **字符串分割 DP 范式**：一维布尔型 dp，`dp[i]` 表示前缀能否被拆分
- **剪枝技巧**：子串长度超过字典最长单词时直接跳过，将 O(n³) 降为 O(n·m²)
- **拆分思维**：判断大问题能否分解为子问题，类似于"完全背包是否存在"的布尔版本
- **substr(pos, len)**：第二个参数是长度不是结束位置，初学者易混淆

### ⚠️ 易错点 & 反思
- ⚠️ **dp 数组大小**：dp[n+1]，下标对应"前 i 个字符"，返回 dp[n] 而不是 dp[n-1]
- ⚠️ **循环边界**：外层 `i <= n`，内层 `j < i`，`substr(j, i-j)` 三个边界要一致
- ⚠️ **set.find 返回值**：返回迭代器，要写 `!= set.end()`，不能直接转 bool
- ✨ 与 LCS 对比：LCS 是"找最长"，Word Break 是"能否拆分"——前者最值型，后者存在型

## [Easy] 买卖股票的最佳时机 (121)

### 🎯 核心思路
一次遍历，两个变量：`minPrice` 记录历史最低价，`maxProfit` 记录截至今天的最大利润。本质是贪心/状态追踪——把暴力枚举 O(n²) 降为 O(n)。

### 🔑 关键代码片段
```cpp
int maxProfit(vector<int>& prices) {
    int minPrice = INT_MAX, maxProfit = 0;
    for (int price : prices) {
        minPrice = min(minPrice, price);
        maxProfit = max(maxProfit, price - minPrice);
    }
    return maxProfit;
}
```

### 📚 学到的知识点
- **状态追踪模式**：用变量维护"到目前为止的最优状态"，一次遍历解决
- **反例思维**：`minPrice` 初始化为 0 是错误的"零成本幻觉"——必须初始化为 INT_MAX 或 prices[0]
- **后验视角**：遍历时虽然不知道未来价格，但知道过去的最低点——"已知过去求当下最值"

### ⚠️ 易错点 & 反思
- ⚠️ `minPrice` 不能初始化为 0（股票价格 > 0），要用 `INT_MAX`
- ⚠️ `minPrice` 和 `maxProfit` 的更新顺序：先更新 min 再计算 profit，不能反过来
- ✨ 追问：如果允许多次买卖（122）或最多两次（123）怎么办？这就是状态机 DP 的入口

## [Medium] 买卖股票的最佳时机 II (122)

### 🎯 核心思路
**贪心法**：累计所有正价差，`Σmax(0, prices[i] - prices[i-1])`。等价于每天都判断"今天比昨天贵就昨天买今天卖"。

### 🔑 关键代码片段
```cpp
int maxProfit(vector<int>& prices) {
    int sum = 0;
    for (int i = 1; i < prices.size(); i++)
        sum += max(0, prices[i] - prices[i-1]);
    return sum;
}
```

### 📚 学到的知识点
- **贪心 vs 状态机 DP**：无限次交易可用贪心（O(n)），有限次交易必须用状态机 DP
- **相邻价差思维**：总利润 = 所有上涨波段的价差之和，不关心具体买卖日
- **状态机模板**（预埋）：`hold = max(hold, notHold - price)` / `notHold = max(notHold, hold + price)`，用于 123/309/714

### ⚠️ 易错点 & 反思
- ⚠️ `prePrice` 必须初始化为 `prices[0]`，未初始化是未定义行为
- ⚠️ 累加用 `+=` 不是 `=`，否则只存了最后一次差价
- ⚠️ 状态机版本要注意**同时更新问题** — 用临时变量存旧 hold
- ✨ 121→122：从"一次交易"到"无限次"，贪心视角完全不同
