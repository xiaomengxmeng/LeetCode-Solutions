## [Easy] Symmetric Tree (101)

### 🎯 核心思路
对称 = 左右子树互为镜像。比较时交叉：`p->left` vs `q->right`，`p->right` vs `q->left`。与 Same Tree 的区别在于"镜像比较"。

### 🔑 关键代码片段
```cpp
// 递归版本
bool compare(TreeNode* p, TreeNode* q) {
    if (!p && !q) return true;
    if (!p || !q) return false;
    return p->val == q->val 
        && compare(p->left, q->right)   // 左 vs 右
        && compare(p->right, q->left);  // 右 vs 左
}

// 迭代版本
bool isSymmetric(TreeNode* root) {
    stack<pair<TreeNode*, TreeNode*>> st;
    st.push({root->left, root->right});
    while (!st.empty()) {
        auto [p, q] = st.top(); st.pop();
        if (!p && !q) continue;
        if (!p || !q) return false;
        if (p->val != q->val) return false;
        st.push({p->left, q->right});   // 交叉比较
        st.push({q->left, p->right});
    }
    return true;
}
```

### 📚 学到的知识点
- **镜像比较**：对称树的关键是交叉比较
- **与 Same Tree 的区别**：Same Tree 同向比较，Symmetric Tree 交叉比较
- **递归转迭代**：用栈模拟递归过程

### ⚠️ 易错点 & 反思
- ❌ 按同向比较（和 Same Tree 一样）
- ✅ 交叉比较：左 vs 右，右 vs 左
- 💡 **技巧**：想象照镜子，左右是反的

## [Easy] Invert Binary Tree (226)

### 🎯 核心思路
递归交换每个节点的左右子树。可以用先序或后序，结果相同。swap 交换的是指针的值（指向的地址）。

### 🔑 关键代码片段
```cpp
// 递归版本
TreeNode* invertTree(TreeNode* root) {
    if (!root) return nullptr;
    swap(root->left, root->right);
    invertTree(root->left);
    invertTree(root->right);
    return root;
}

// 迭代版本（栈）
TreeNode* invertTree(TreeNode* root) {
    if (!root) return nullptr;
    stack<TreeNode*> st;
    st.push(root);
    while (!st.empty()) {
        TreeNode* node = st.top(); st.pop();
        swap(node->left, node->right);
        if (node->left) st.push(node->left);
        if (node->right) st.push(node->right);
    }
    return root;
}
```

### 📚 学到的知识点
- **swap 原理**：交换两个指针指向的地址
- **翻转本质**：每个节点的左右子树交换
- **递归顺序无关**：先序/后序结果相同

### ⚠️ 易错点 & 反思
- ❌ 忘记处理空节点
- ✅ 空节点返回 nullptr
- 💡 **技巧**：想象照镜子，左右互换
