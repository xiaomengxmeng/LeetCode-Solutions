## [Easy] Symmetric Tree (101)

### 🎯 核心思路
对称 = 左右子树互为镜像。比较时交叉：`p->left` vs `q->right`，`p->right` vs `q->left`。与 Same Tree 的区别在于"镜像比较"。

### 🔑 关键代码片段
```cpp
// 递归版本
bool compare(TreeNode* p, TreeNode* q) {
    if (!p && !q) return true;
    if (!p || !q) return false;
    return p->val == q->val 
        && compare(p->left, q->right)   // 左 vs 右
        && compare(p->right, q->left);  // 右 vs 左
}

// 迭代版本
bool isSymmetric(TreeNode* root) {
    stack<pair<TreeNode*, TreeNode*>> st;
    st.push({root->left, root->right});
    while (!st.empty()) {
        auto [p, q] = st.top(); st.pop();
        if (!p && !q) continue;
        if (!p || !q) return false;
        if (p->val != q->val) return false;
        st.push({p->left, q->right});   // 交叉比较
        st.push({q->left, p->right});
    }
    return true;
}
```

### 📚 学到的知识点
- **镜像比较**：对称树的关键是交叉比较
- **与 Same Tree 的区别**：Same Tree 同向比较，Symmetric Tree 交叉比较
- **递归转迭代**：用栈模拟递归过程

### ⚠️ 易错点 & 反思
- ❌ 按同向比较（和 Same Tree 一样）
- ✅ 交叉比较：左 vs 右，右 vs 左
- 💡 **技巧**：想象照镜子，左右是反的

## [Easy] Invert Binary Tree (226)

### 🎯 核心思路
递归交换每个节点的左右子树。可以用先序或后序，结果相同。swap 交换的是指针的值（指向的地址）。

### 🔑 关键代码片段
```cpp
// 递归版本
TreeNode* invertTree(TreeNode* root) {
    if (!root) return nullptr;
    swap(root->left, root->right);
    invertTree(root->left);
    invertTree(root->right);
    return root;
}

// 迭代版本（栈）
TreeNode* invertTree(TreeNode* root) {
    if (!root) return nullptr;
    stack<TreeNode*> st;
    st.push(root);
    while (!st.empty()) {
        TreeNode* node = st.top(); st.pop();
        swap(node->left, node->right);
        if (node->left) st.push(node->left);
        if (node->right) st.push(node->right);
    }
    return root;
}
```

### 📚 学到的知识点
- **swap 原理**：交换两个指针指向的地址
- **翻转本质**：每个节点的左右子树交换
- **递归顺序无关**：先序/后序结果相同

### ⚠️ 易错点 & 反思
- ❌ 忘记处理空节点
- ✅ 空节点返回 nullptr
- 💡 **技巧**：想象照镜子，左右互换

## [Medium] Binary Tree Level Order Traversal (102)

### 🎯 核心思路
BFS 层序遍历。用队列，一次处理一整层。关键是用 `size` 记录当前层节点数，确保不会处理到下一层的节点。

### 🔑 关键代码片段
```cpp
vector<vector<int>> levelOrder(TreeNode* root) {
    vector<vector<int>> res;
    if (!root) return res;
    queue<TreeNode*> q;
    q.push(root);
    while (!q.empty()) {
        int size = q.size();  // 当前层节点数
        vector<int> level;
        for (int i = 0; i < size; i++) {
            TreeNode* node = q.front(); q.pop();
            level.push_back(node->val);
            if (node->left) q.push(node->left);
            if (node->right) q.push(node->right);
        }
        res.push_back(level);
    }
    return res;
}
```

### 📚 学到的知识点
- **BFS 层序遍历**：用队列，先进先出
- **一次处理一层**：用 `size` 记录当前层节点数
- **DFS 也可以**：传递 depth 参数

### ⚠️ 易错点 & 反思
- ❌ 不记录 size，导致队列中混合两层节点
- ✅ 用 `for (int i = 0; i < size; i++)` 确保只处理当前层
- 💡 **技巧**：`size` 必须在 for 循环外获取，因为队列大小会变
