## \[Easy] Next Greater Element I (496)

### 🎯 核心思路

用单调递减栈预处理 nums2，找出每个元素的"下一个更大元素"，存入 map。然后遍历 nums1 直接查表得到答案。时间复杂度 O(n + m)。

### 🔑 关键代码片段

```cpp
vector<int> nextGreaterElement(vector<int>& nums1, vector<int>& nums2) {
    unordered_map<int, int> greater_map;
    stack<int> st;
    for (int x : nums2) {
        while (!st.empty() && x > st.top()) {
            greater_map[st.top()] = x;
            st.pop();
        }
        st.push(x);
    }
    vector<int> res;
    for (int x : nums1) {
        if (greater_map.count(x)) {
            res.push_back(greater_map[x]);
        } else {
            res.push_back(-1);
        }
    }
    return res;
}
```

### 📚 学到的知识点

- **单调栈**：栈内元素保持单调递减，用于找"下一个更大元素"
- **预处理思想**：先算出所有答案，再查表
- **map 查询**：用 `count()` 判断 key 是否存在

### ⚠️ 易错点 & 反思

- ❌ `greater_map(x)` 语法错误，map 用 `[]` 访问
- ❌ 不能用栈顶判断是否有答案，栈里可能剩多个元素
- ✅ 用 `greater_map.count(x)` 判断是否存在答案

## [Medium] Daily Temperatures (739)

### 🎯 核心思路
用单调递减栈存下标，遍历温度数组。当遇到更高温度时，栈顶元素找到答案，计算天数差值。栈内剩余元素答案为 0。时间复杂度 O(n)。

### 🔑 关键代码片段
```cpp
vector<int> dailyTemperatures(vector<int>& temperatures) {
    vector<int> res(temperatures.size(), 0);
    stack<int> st;
    for (int i = 0; i < temperatures.size(); i++) {
        while (!st.empty() && temperatures[i] > temperatures[st.top()]) {
            int top = st.top();
            st.pop();
            res[top] = i - top;
        }
        st.push(i);
    }
    return res;
}
```

### 📚 学到的知识点
- **单调栈存下标**：需要位置信息时存下标，不是元素值
- **结果数组初始化为 0**：栈里剩余元素自动答案为 0
- **不修改原数组**：创建新数组存储结果

### ⚠️ 易错点 & 反思
- ❌ 修改了原数组，应该创建新数组
- ❌ 忘记处理栈里剩余元素（应该返回 0）
- ✅ 初始化 `res(n, 0)` 自动处理剩余元素
