## [Easy] Convert Sorted Array to Binary Search Tree (108)

### 🎯 核心思路
升序数组 → 高度平衡 BST = **二分递归法**。每次取数组中间元素作为根节点，左半部分构建左子树，右半部分构建右子树。与二分查找同源。

### 🔑 关键代码片段
```cpp
TreeNode* buildTree(vector<int>& nums, int left, int right) {
    if (left > right) return nullptr;
    int mid = left + (right - left) / 2;
    TreeNode* root = new TreeNode(nums[mid]);
    root->left = buildTree(nums, left, mid - 1);
    root->right = buildTree(nums, mid + 1, right);
    return root;
}
```

### 📚 学到的知识点
- **BST + 有序数组**：有序数组的中间节点 = BST 的根节点
- **递归构建模板**：选根 → 左区间递归 → 右区间递归
- **`mid = left + (right - left) / 2`**：防止 left+right 溢出
- **递归终止条件**：`left > right` 返回 nullptr

### ⚠️ 易错点 & 反思
- ❌ 忘记递归终止条件 → 无限递归，栈溢出
- ❌ `mid` 计算时直接用 `(left+right)/2` → left+right 可能溢出
- ✅ 用 `left + (right - left) / 2` 更加安全
- 💡 核心模式：**二分+分治**，与二分查找是同源思想

## [Medium] Kth Smallest Element in a BST (230)

### 🎯 核心思路
BST 中序遍历 = **递增序列**，第 k 小 = 中序序列第 k 个元素。用**迭代中序遍历**依次取出节点值，取 `vals[k-1]`。

### 🔑 关键代码片段
```cpp
int kthSmallest(TreeNode* root, int k) {
    vector<int> vals;
    stack<TreeNode*> st;
    TreeNode* cur = root;
    while (!st.empty() || cur) {
        while (cur) {
            st.push(cur);
            cur = cur->left;
        }
        cur = st.top();
        vals.push_back(cur->val);
        st.pop();
        cur = cur->right;
    }
    return vals[k - 1];
}
```

### 📚 学到的知识点
- **BST 性质**：中序遍历 = 升序序列，第 k 个即第 k 小
- **迭代中序模板**：左链入栈 → 出栈访问 → 切右子树
- **进阶优化**：不需要存完整数组，找到第 k 个直接返回，提前终止

### ⚠️ 易错点 & 反思
- ✅ 中序记住：**左-根-右**，栈模拟的关键是"先一路向左走到底"
- 💡 **优化点**：可以边遍历边计数，`count == k` 时直接 return，避免遍历整棵树
- 💡 当前解法 O(n) 空间存数组，优化到 O(1) 空间

## [Medium] Lowest Common Ancestor of a BST (235)

### 🎯 核心思路
利用 BST 性质：`左<根<右`。若 p、q 都在右侧 → 递归右子树；都在左侧 → 递归左子树；分居两侧 → 当前根即为 LCA。

### 🔑 关键代码片段
```cpp
TreeNode* lowestCommonAncestor(TreeNode* root, TreeNode* p, TreeNode* q) {
    if (root->val < p->val && root->val < q->val)
        return lowestCommonAncestor(root->right, p, q);
    else if (root->val > p->val && root->val > q->val)
        return lowestCommonAncestor(root->left, p, q);
    else
        return root;
}
```

### 📚 学到的知识点
- **BST 的 LCA 比普通二叉树简单**：普通二叉树需要后序遍历，BST 只需比较值
- **判断标准**：p 和 q 分居 root 两侧 → root 即为 LCA
- **递归方向**：都在右 → 往右走；都在左 → 往左走
- **迭代版本**：可将递归改为 while 循环，O(1) 空间

### ⚠️ 易错点 & 反思
- ✅ 条件判断用 `root->val < p->val && root->val < q->val`，不是 `p->val < root->val`
- 💡 **迭代优化**：可以改成 while 循环，不用递归栈，空间 O(1)
- 💡 这题是 **LCA 系列的核心基础**：BST → 带父指针 → 普通二叉树
