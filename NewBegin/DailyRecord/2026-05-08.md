## [Easy] Minimum Depth of Binary Tree (111)

### 🎯 核心思路
最小深度不能直接用 min(左, 右) + 1，因为空子树没有叶子节点。正确逻辑：如果一边为空，只看另一边；如果两边都不为空，才用 min。BFS 更适合，找到第一个叶子节点就返回。

### 🔑 关键代码片段
```cpp
// DFS 递归
int minDepth(TreeNode* root) {
    if (!root) return 0;
    if (!root->left && !root->right) return 1;  // 叶子节点
    if (!root->left) return minDepth(root->right) + 1;  // 只有右子树
    if (!root->right) return minDepth(root->left) + 1;  // 只有左子树
    return min(minDepth(root->left), minDepth(root->right)) + 1;
}

// BFS 迭代（更优）
int minDepth(TreeNode* root) {
    if (!root) return 0;
    queue<TreeNode*> q;
    q.push(root);
    int depth = 0;
    while (!q.empty()) {
        depth++;
        int size = q.size();
        for (int i = 0; i < size; i++) {
            TreeNode* node = q.front(); q.pop();
            if (!node->left && !node->right) return depth;  // 找到叶子！
            if (node->left) q.push(node->left);
            if (node->right) q.push(node->right);
        }
    }
    return depth;
}
```

### 📚 学到的知识点
- **不能直接用 min**：空子树没有叶子节点
- **叶子节点定义**：没有左右子树的节点
- **BFS 更适合最小深度**：找到第一个叶子就返回

### ⚠️ 易错点 & 反思
- ❌ 直接用 `min(左, 右) + 1`
- ✅ 先判断是否有一边为空
- 💡 **技巧**：BFS 找最小深度，DFS 找最大深度
