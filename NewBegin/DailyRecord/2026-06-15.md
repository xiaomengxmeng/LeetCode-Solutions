## [Medium] Partition Equal Subset Sum (416)
### 🎯 核心思路
- 问题转化：判断是否能选出若干元素，使其和等于 sum/2 → 0-1背包
- 剪枝：总和为奇数 → 直接返回 false
- 状态定义：`dp[i][j]` = 从前 i 个元素中选，能否恰好凑出总和 j
- 转移：`dp[i][j] = dp[i-1][j] || dp[i-1][j - nums[i-1]]`（选或不选）
- 边界：`dp[0][0] = true`，`dp[i][0] = true`

### 🔑 关键代码片段
```cpp
dp[0][0] = true;
for (int i = 1; i <= n; i++) {
    dp[i][0] = true;
    for (int j = 1; j <= target; j++) {
        dp[i][j] = dp[i - 1][j];
        if (j >= nums[i - 1])
            dp[i][j] = dp[i][j] || dp[i - 1][j - nums[i - 1]];
    }
}
return dp[n][target];
```

### 📚 学到的知识点
- 0-1背包 DP 基本模板：状态定义 + 选/不选转移
- 问题转化能力：分割等和子集 → 子集和问题 → 背包问题
- 空间可以优化：二维 → 一维（倒序遍历），但初次学习先掌握二维

### ⚠️ 易错点 & 反思
- ⚠️ **边界一致性**：dp 数组用 `n+1` 行时，遍历到 `i <= n`，取 nums[i-1]
- ⚠️ 忘记判断 sum 是否为奇数就直接算 target
- ⚠️ dp 初始化：`dp[0][0] = true` 是起点，`dp[i][0] = true` 是每行边界
