## [Easy] Binary Tree Preorder Traversal (144)

### 🎯 核心思路
递归实现前序遍历（根→左→右）。先处理根节点，再递归左子树，最后递归右子树。关键是将左右子树的结果合并到当前 vector 中。

### 🔑 关键代码片段
```cpp
vector<int> preorderTraversal(TreeNode *root) {
    vector<int> vec;
    if (root == nullptr)
        return vec;
    vec.push_back(root->val);
    vector<int> left = preorderTraversal(root->left);
    vector<int> right = preorderTraversal(root->right);
    vec.insert(vec.end(), left.begin(), left.end());
    vec.insert(vec.end(), right.begin(), right.end());
    return vec;
}
```

### 📚 学到的知识点
- **前序遍历顺序**：根 → 左 → 右
- **insert 合并 vector**：`vec.insert(vec.end(), v.begin(), v.end())`
- **迭代思路**：栈维护，先压右再压左（保证左先出栈）
- **三种遍历关系**：前序(根左右)、中序(左根右)、后序(左右根)

### ⚠️ 易错点 & 反思
- ❌ `.begin()` 和 `.end()` 各调用一次递归 → 会导致重复遍历，O(2ⁿ)
- ✅ 先保存左/右子树结果到变量，再 insert
- ❌ 忘记空节点判断 → 必须 `if (root == nullptr) return vec`

## [Easy] Binary Tree Inorder Traversal (94)

### 🎯 核心思路
迭代实现中序遍历（左→根→右）。一路向左将所有节点入栈，弹出时处理值，再转向右子树。外层循环条件：`cur != nullptr || !st.empty()` 确保所有节点都能被访问。

### 🔑 关键代码片段
```cpp
vector<int> inorderTraversal(TreeNode* root) {
    stack<TreeNode*> st;
    vector<int> vec;
    TreeNode *cur = root;
    while(cur!=nullptr||!st.empty())
    {
        while(cur!=nullptr)
        {
            st.push(cur);
            cur = cur->left;
        }
        cur = st.top();
        st.pop();
        vec.push_back(cur->val);
        cur = cur->right;
    }
    return vec;
}
```

### 📚 学到的知识点
- **中序迭代模板**：一路向左入栈 → 弹出处理 → 转向右
- **前序 vs 中序迭代**：
  - 前序：处理根 → 压右 → 压左（或入栈时处理）
  - 中序：一路向左 → 弹出才处理
- **外层循环条件**：`cur || stack` 缺一不可

### ⚠️ 易错点 & 反思
- ❌ 外层循环只写 `cur != nullptr` → 栈还有节点但 cur 为空时会提前退出
- ✅ 必须 `cur != nullptr || !st.empty()`
- ❌ 内层循环忘记入栈 → 丢失中间节点
- ✅ 先入栈再 `cur = cur->left`

## [Easy] Binary Tree Postorder Traversal (145)

### 🎯 核心思路
利用前序迭代的变体：先交换左右入栈顺序得到"根→右→左"，再 reverse 得到后序"左→右→根"。

### 🔑 关键代码片段
```cpp
vector<int> postorderTraversal(TreeNode *root) {
    vector<int> vec;
    if (!root) return vec;
    stack<TreeNode *> st;
    st.push(root);
    while (!st.empty()) {
        TreeNode *cur = st.top(); st.pop();
        vec.push_back(cur->val);
        if (cur->left)  st.push(cur->left);
        if (cur->right) st.push(cur->right);
    }
    reverse(vec.begin(), vec.end());
    return vec;
}
```

### 📚 学到的知识点
- **后序 = 前序变体 + reverse**：根→右→左 反转得 左→右→根
- **入栈顺序决定处理顺序**：栈是 LIFO，先压的后出
- **三种遍历迭代全部掌握**：前序(栈遍历)、中序(一路向左)、后序(reverse)
- 别忘了判空：`if (!root) return vec;`

### ⚠️ 易错点 & 反思
- ❌ 入栈顺序搞反 → 先压左再压右出的是"根→左→右"，不是后序
- ✅ 先压左再压右 + reverse = 后序
- ❌ 忘记判空 → 空指针崩溃
- ❌ 把后序的"左右根"直接当成了中序来写 → 混淆了遍历顺序

## [Medium] Validate Binary Search Tree (98)

### 🎯 核心思路
利用 BST 中序遍历严格递增的性质。复用中序迭代模板，用 prev 指针记录上一个弹出的节点。当前节点值必须 > prev 的值，否则不是 BST。prev 首次不比较，且只能在弹出节点时更新（不能在入栈前初始化）。

### 🔑 关键代码片段
```cpp
bool isValidBST(TreeNode* root) {
    stack<TreeNode*> st;
    TreeNode* cur = root;
    TreeNode* prev = nullptr;
    while (!st.empty() || cur) {
        while (cur) {
            st.push(cur);
            cur = cur->left;
        }
        cur = st.top(); st.pop();
        if (prev && cur->val <= prev->val)
            return false;
        prev = cur;
        cur = cur->right;
    }
    return true;
}
```

### 📚 学到的知识点
- **BST 性质**：中序遍历结果严格递增
- **中序模板迁移**：98 题 = 94 题模板 + prev 比较
- **prev 初始化**：nullptr 首次不比较，弹出时才更新

### ⚠️ 易错点 & 反思
- ❌ `prev` 在入栈前初始化 → 指向了根节点而非中序第一个节点
- ✅ `prev` 在弹出时更新，且 `if(prev && ...)` 跳过首次比较
- ❌ 忘记了 BST 的"严格递增" → 必须用 `<=` 返回 false，不能用 `<`
