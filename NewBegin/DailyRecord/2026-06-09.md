## [Medium] 不同路径 (62)

### 🎯 核心思路
二维 DP 入门：dp[i][j] 表示走到 (i, j) 的路径数。第一行/列只有一种走法，其他格从上方或左边过来。

### 🔑 关键代码片段
```cpp
int uniquePaths(int m, int n) {
    vector<vector<int>> dp(m, vector<int>(n, 0));
    for (int i = 0; i < m; i++) dp[i][0] = 1;
    for (int j = 0; j < n; j++) dp[0][j] = 1;
    for (int i = 1; i < m; i++)
        for (int j = 1; j < n; j++)
            dp[i][j] = dp[i-1][j] + dp[i][j-1];
    return dp[m-1][n-1];
}
```

### 📚 学到的知识点
- **二维 DP 模板**：dp[i][j] + 边界初始化 + 双循环
- **边界思维**：第一行只能从左边来（1种），第一列只能从上面来（1种）
- **空间优化**：可降为一维 dp[j] += dp[j-1]

### ⚠️ 易错点 & 反思
- ❌ 转移方程误加 +1：`dp[i][j] = dp[i-1][j] + dp[i][j-1]` 就对了，不需要 +1
- dp 大小是 m×n，返回 dp[m-1][n-1]
- 与 70 题本质相同：都是求和，只是 70 是一维，62 是二维

### 🔗 关联题目
- 70. Climbing Stairs（一维版本）
- 63. Unique Paths II（加障碍物）

## [Medium] 不同路径 II (63)

### 🎯 核心思路
62 的障碍物版本：障碍物位置 dp[i][j] = 0，边界遇障碍物用 break 截断，起点/终点障碍物直接 return 0。

### 🔑 关键代码片段
```cpp
int uniquePathsWithObstacles(vector<vector<int>>& obstacleGrid) {
    int m = obstacleGrid.size(), n = obstacleGrid[0].size();
    if (obstacleGrid[0][0] == 1 || obstacleGrid[m-1][n-1] == 1) return 0;
    vector<vector<int>> dp(m, vector<int>(n, 0));
    for (int i = 0; i < m; i++) {
        if (obstacleGrid[i][0]) break;  // 障碍物后全为0
        dp[i][0] = 1;
    }
    for (int j = 0; j < n; j++) {
        if (obstacleGrid[0][j]) break;  // 障碍物后全为0
        dp[0][j] = 1;
    }
    for (int i = 1; i < m; i++)
        for (int j = 1; j < n; j++)
            if (obstacleGrid[i][j])
                dp[i][j] = 0;
            else
                dp[i][j] = dp[i-1][j] + dp[i][j-1];
    return dp[m-1][n-1];
}
```

### 📚 学到的知识点
- **障碍物思维**：遇到障碍 = 路径数为 0
- **break 截断**：第一行/列遇到第一个障碍物，后面全部不可达
- **起点/终点特判**：起终点被堵直接返回 0

### ⚠️ 易错点 & 反思
- 起点/终点障碍物检查必须放前面，避免后续计算
- 边界用 break 比用 if 逐个判断更简洁
- 与 62 的关系：62 是 63 的无障碍版本

### 🔗 关联题目
- 62. Unique Paths（无障碍版本）
- 64. Minimum Path Sum（加权重变体）

## [Medium] 完全平方数 (279)

### 🎯 核心思路
一维最值型 DP：dp[i] 表示凑成 i 需要的最少完全平方数个数。对每个 i，尝试减去所有可能的平方数 j²，取最小值。本质和爬楼梯一样是"减一个值看之前的状态"，但选择空间更大（√n 种），且从求和变成了求最小值。

### 🔑 关键代码片段
```cpp
int numSquares(int n) {
    vector<int> dp(n+1, 0);
    for (int i = 1; i <= n; i++) {
        dp[i] = i;  // 最坏情况：全用 1²
        for (int j = 1; j * j <= i; j++) {
            dp[i] = min(dp[i], dp[i - j*j] + 1);
        }
    }
    return dp[n];
}
```

### 📚 学到的知识点
- **最值型 DP 模板**：dp[i] = min/max over 所有选择 (dp[i - 选择] + 代价)
- **初始化为上界**：最值问题中 dp[i] 先设为可能的最大值（这里是 i），再不断取小
- **"减一个值"递推结构**：70 减 1/2 → 279 减 j²，框架一致，选择空间不同

### ⚠️ 易错点 & 反思
- ❌ 内层循环条件用 `<` 而不是 `<=`，会漏掉 i 本身就是完美平方数的最优解（dp[4] = 1）
- 必须遍历所有 j² ≤ i，不只是部分平方数
- dp[0] = 0 作为 base case，当 i = j² 时走 dp[0] + 1 = 1

### 🔗 关联题目
- 70. Climbing Stairs（同结构，求和型 + 固定选择）
- 322. Coin Change（同类：最值型 DP，选择硬币面额）
