## [Easy] Maximum Depth of Binary Tree (104)

### 🎯 核心思路
DFS 递归：树的深度 = max(左子树深度, 右子树深度) + 1。空节点深度为 0。时间复杂度 O(n)，空间复杂度 O(h)。

### 🔑 关键代码片段
```cpp
int maxDepth(TreeNode* root) {
    if (!root) return 0;
    return max(maxDepth(root->left), maxDepth(root->right)) + 1;
}
```

### 📚 学到的知识点
- **DFS 递归**：树的问题天然适合递归
- **深度公式**：max(左, 右) + 1
- **BFS 层序遍历**：也可以用来求深度

### ⚠️ 易错点 & 反思
- ❌ 忘记处理空节点
- ✅ 空节点返回 0
- 💡 **DFS vs BFS**：DFS 代码简洁，BFS 空间稳定
