## [Easy] Maximum Depth of Binary Tree (104)

### 🎯 核心思路
DFS 递归：树的深度 = max(左子树深度, 右子树深度) + 1。空节点深度为 0。时间复杂度 O(n)，空间复杂度 O(h)。

### 🔑 关键代码片段
```cpp
int maxDepth(TreeNode* root) {
    if (!root) return 0;
    return max(maxDepth(root->left), maxDepth(root->right)) + 1;
}
```

### 📚 学到的知识点
- **DFS 递归**：树的问题天然适合递归
- **深度公式**：max(左, 右) + 1
- **BFS 层序遍历**：也可以用来求深度

### ⚠️ 易错点 & 反思
- ❌ 忘记处理空节点
- ✅ 空节点返回 0
- 💡 **DFS vs BFS**：DFS 代码简洁，BFS 空间稳定

## [Easy] Same Tree (100)

### 🎯 核心思路
DFS 递归比较两棵树。终止条件：都为空返回 true，只有一个为空返回 false，值不等返回 false。然后递归比较左右子树。

### 🔑 关键代码片段
```cpp
bool isSameTree(TreeNode* p, TreeNode* q) {
    if (!p && !q) return true;
    if (!p || !q) return false;
    return p->val == q->val && isSameTree(p->left, q->left) && isSameTree(p->right, q->right);
}
```

### 📚 学到的知识点
- **空节点处理**：`!p && !q` 和 `!p || !q` 组合
- **递归比较**：值相同 + 左子树相同 + 右子树相同
- **短路求值**：`&&` 遇到 false 立即返回
- **递归转迭代**：用显式栈/队列模拟递归

### ⚠️ 易错点 & 反思
- ❌ 忘记处理空节点
- ✅ 先处理空节点，再比较值
- 💡 **技巧**：两行代码处理所有空节点情况

### 🔄 递归转迭代模板

**迭代 DFS（栈）**：
```cpp
bool isSameTree(TreeNode* p, TreeNode* q) {
    stack<pair<TreeNode*, TreeNode*>> st;
    st.push({p, q});
    while (!st.empty()) {
        auto [n1, n2] = st.top(); st.pop();
        if (!n1 && !n2) continue;
        if (!n1 || !n2) return false;
        if (n1->val != n2->val) return false;
        st.push({n1->right, n2->right});
        st.push({n1->left, n2->left});
    }
    return true;
}
```

**迭代 BFS（队列）**：
```cpp
bool isSameTree(TreeNode* p, TreeNode* q) {
    queue<pair<TreeNode*, TreeNode*>> qu;
    qu.push({p, q});
    while (!qu.empty()) {
        auto [n1, n2] = qu.front(); qu.pop();
        if (!n1 && !n2) continue;
        if (!n1 || !n2) return false;
        if (n1->val != n2->val) return false;
        qu.push({n1->left, n2->left});
        qu.push({n1->right, n2->right});
    }
    return true;
}
```

**转换要点**：
| 递归 | 迭代 |
|------|------|
| 终止条件 | 循环中 continue/return |
| 递归调用 | push 到栈/队列 |
| 系统栈 | 显式栈/队列 |
