## [Hard] Trapping Rain Water (42)

### 🎯 核心思路
单调递增栈按行计算雨水。当遇到比栈顶高的柱子时，栈顶是凹槽底部，新栈顶是左边界，当前元素是右边界。水量 = 宽度 × 高度，其中高度 = min(左边界, 右边界) - 底部。

### 🔑 关键代码片段
```cpp
int trap(vector<int>& height) {
    stack<int> st;
    int n = height.size();
    int res = 0;
    for (int i = 0; i < n; i++) {
        while (!st.empty() && height[st.top()] < height[i]) {
            int top = st.top();
            st.pop();
            if (st.empty()) break;
            res += (i - st.top() - 1) * (min(height[st.top()], height[i]) - height[top]);
        }
        st.push(i);
    }
    return res;
}
```

### 📚 学到的知识点
- **单调栈按行计算**：栈顶是底部，新栈顶是左边界，当前是右边界
- **水高度公式**：`min(左边界高度, 右边界高度) - 底部高度`
- **水宽度公式**：`右边界 - 左边界 - 1`

### ⚠️ 易错点 & 反思
- ❌ 水高度只用了右边界高度，没用 min(左, 右)
- ✅ 水高度 = min(height[left], height[i]) - height[bottom]
- 💡 **关键理解**：凹槽的左右边界是"第一个比它高的"，不是"最高的"
