# 📚 2026-06-01 周复习 Day 1 — 数组 + 链表

## [Easy] Two Sum (1)
### 🎯 核心思路
哈希表一次遍历：存已遍历元素的下标，检查 `target - nums[i]` 是否在表中。
### 🔑 关键代码片段
```cpp
unordered_map<int, int> map;
for (int i = 0; i < nums.size(); i++) {
    int targetNum = target - nums[i];
    if (map.find(targetNum) != map.end())
        return {map[targetNum], i};
    map[nums[i]] = i;
}
return {};
```
### 📚 对比原解法
**一致**——和 04-13 的写法完全相同，一次过 ✅
### ⚠️ 易错点 & 反思
- 先判断再插入，避免自身匹配

## [Medium] Longest Substring Without Repeating Characters (3)
### 🎯 核心思路
滑动窗口：右指针扩展窗口，出现重复时 while 收缩左指针直到无重复，每次更新最大长度。
### 🔑 关键代码片段
```cpp
vector<int> freq(128, 0);
int left = 0, ans = 0;
for (int right = 0; right < s.size(); right++) {
    freq[s[right]]++;
    while (freq[s[right]] > 1)
        freq[s[left++]]--;
    ans = max(ans, right - left + 1);
}
return ans;
```
### 📚 对比原解法
**一致**——原解法用的是存下标版（`freq[c]>=left` 直接跳），这次写了频次版（while 一步步移），思路等价 ✅
### ⚠️ 易错点 & 反思
- 频次版比存下标版更容易理解，但需要 while 循环多移几步
- 两种写法都要注意数组大小 128（ASCII）

## [Medium] Merge Intervals (56)
### 🎯 核心思路
按 start 排序后遍历，重叠则合并右边界，不重叠则加入结果。
### 🔑 关键代码片段
```cpp
sort(intervals.begin(), intervals.end());
vector<int> cur = intervals[0];
for (int i = 1; i < intervals.size(); i++) {
    if (intervals[i][0] <= cur[1])
        cur[1] = max(intervals[i][1], cur[1]);
    else {
        res.push_back(cur);
        cur = intervals[i];
    }
}
res.push_back(cur);
```
### 📚 对比原解法
**一致**——排序 + 遍历合并 ✅
### ⚠️ 易错点 & 反思
- ❌ 空数组检查放到了 cur=intervals[0] 之后 → 先判空再取值

## [Easy] Move Zeroes (283)
### 🎯 核心思路
双指针：fast 遍历，非零则赋值给 slow 并 slow++；遍历完后 slow 后面全填 0。
### 🔑 关键代码片段
```cpp
int slow = 0;
for (int fast = 0; fast < nums.size(); fast++)
    if (nums[fast] != 0) nums[slow++] = nums[fast];
while (slow < nums.size()) nums[slow++] = 0;
```
### 📚 对比原解法
**一致**——双指针标准写法 ✅
### ⚠️ 易错点 & 反思
- while 填 0 的起始位置是 slow，不是 0

## [Easy] Reverse Linked List (206)
### 🎯 核心思路
三指针：存 next → 指 prev → prev 前移 → cur 前移。
### 🔑 关键代码片段
```cpp
ListNode* prev = nullptr, *cur = head;
while (cur) {
    ListNode* next = cur->next;
    cur->next = prev;
    prev = cur;
    cur = next;
}
return prev;
```
### 📚 对比原解法
**进步**——上次复习 5 月 22 日第一次写错了（写成对交换），这次一次过 ✅
### ⚠️ 易错点 & 反思
- 核心口诀："先存 next，再指 prev，一起走"

## [Medium] Linked List Cycle II (142)
### 🎯 核心思路
两阶段法：快慢相遇确定有环 → 从头和相遇点同步走，相遇处即入环点。
### 🔑 关键代码片段
```cpp
ListNode *slow = head, *fast = head;
while (fast && fast->next) {
    slow = slow->next;
    fast = fast->next->next;
    if (slow == fast) break;
}
if (!fast || !fast->next) return nullptr;
ListNode *p1 = head, *p2 = slow;
while (p1 != p2) { p1 = p1->next; p2 = p2->next; }
return p1;
```
### 📚 对比原解法
**一致**——两阶段法 ✅
### ⚠️ 易错点 & 反思
- 无环判断：`!fast || !fast->next`

## [Easy] Palindrome Linked List (234)
### 🎯 核心思路
找中点 → 反转后半 → 比较两段。
### 🔑 关键代码片段
```cpp
ListNode *slow = head, *fast = head;
while (fast && fast->next) { fast = fast->next->next; slow = slow->next; }
ListNode *prev = nullptr, *cur = slow;
while (cur) {  // 反转后半
    ListNode *next = cur->next;
    cur->next = prev;
    prev = cur;
    cur = next;
}
// 比较
while (right) {
    if (left->val != right->val) return false;
    left = left->next; right = right->next;
}
```
### 📚 对比原解法
**一致**✅
### ⚠️ 易错点 & 反思
- ❌ `while (cur->next)` → 最后一个节点没反转；✅ `while (cur)`

## [Medium] Remove Nth Node From End (19)
### 🎯 核心思路
dummy 头节点 + 快慢指针间隔 n 步。
### 🔑 关键代码片段
```cpp
ListNode* dummy = new ListNode(0, head);
ListNode *slow = dummy, *fast = dummy;
for (int i = 0; i < n; i++) fast = fast->next;
while (fast->next) { slow = slow->next; fast = fast->next; }
slow->next = slow->next->next;
return dummy->next;
```
### 📚 对比原解法
**一致**——和 04-23 写法相同 ✅
### ⚠️ 易错点 & 反思
- dummy 头节点统一处理删除头节点的情况
