## [Medium] Permutations (46)
### 🎯 核心思路
回溯 + used 数组：三个位置依次选择，用 `used[i]` 标记已选元素，到叶子节点（path.size() == nums.size()）记录结果。
### 🔑 关键代码片段
```cpp
vector<bool> used(nums.size(), false);
void backtrack(vector<int>& nums, vector<bool>& used, vector<int>& path, vector<vector<int>>& res) {
    if (path.size() == nums.size()) { res.push_back(path); return; }
    for (int i = 0; i < nums.size(); i++) {
        if (used[i]) continue;
        used[i] = true;
        path.push_back(nums[i]);
        backtrack(nums, used, path, res);
        used[i] = false;
        path.pop_back();
    }
}
```
### 📚 对比原解法
**一致** ✅ — 经典回溯模板完全正确，代码组织结构一致
### ⚠️ 易错点 & 反思
- used 数组要初始化大小：`vector<bool> used(nums.size(), false);`
- 回溯三部曲：① 标记 used ② 递归 ③ 撤销 used
- 与 78（子集）的区别：全排列到叶子才记录，子集每层都记录

## [Medium] Combination Sum (39)
### 🎯 核心思路
回溯 + 排序剪枝：用 `start` 索引防重复组合，可重复选取时传 `i`（而非 `i+1`），排序后 `sum + candidates[i] > target → break` 剪枝。
### 🔑 关键代码片段
```cpp
sort(candidates.begin(), candidates.end());
void backtrack(vector<int>& candidates, int start, int sum, vector<int>& path, vector<vector<int>>& res, int target) {
    if (sum == target) { res.push_back(path); return; }
    for (int i = start; i < candidates.size(); i++) {
        if (sum + candidates[i] > target) break;
        path.push_back(candidates[i]);
        backtrack(candidates, i, sum + candidates[i], path, res, target);
        path.pop_back();
    }
}
```
### 📚 对比原解法
**一致** ✅ — 排序剪枝思路完全正确，传 i 实现可重复选取
### ⚠️ 易错点 & 反思
- 排序是剪枝的前提：只有升序排列才能保证 `candidates[i+1] >= candidates[i]`，遇到超 target 的直接 break
- 可重复选取传 `i`，不可重复传 `i+1`（这是 39 和 40 的核心区别）
- `sum` 传值（无需回溯），`path` 传引用（需 pop_back）
- 可以用 `last` 递减版：`dfs(path, last - candidates[i], candidates, i, res)` — 本质等效

## [Medium] Word Search (79)
### 🎯 核心思路
网格回溯：遍历每格作起点，DFS 四方向搜索。用 `'#'` 标记已访问（临时修改原数组），回溯后恢复。
### 🔑 关键代码片段
```cpp
vector<vector<int>> dirs = {{0,1},{0,-1},{1,0},{-1,0}};
bool dfs(vector<vector<char>>& board, string& word, int k, int i, int j) {
    if (i < 0 || i >= m || j < 0 || j >= n) return false;
    if (board[i][j] != word[k]) return false;
    if (k == word.size() - 1) return true;
    char temp = board[i][j]; board[i][j] = '#';
    for (auto& d : dirs)
        if (dfs(board, word, k + 1, i + d[0], j + d[1])) return true;
    board[i][j] = temp;
    return false;
}
```
### 📚 对比原解法
**一致** ✅ — 模板写法完全一致，方向数组+原数组标记法
### ⚠️ 易错点 & 反思
- k == word.size()-1 为终止条件（已找到完整单词），而非 k == word.size()
- 原数组标记法：省 visited 数组，但需 char temp 暂存 + 回溯恢复
- 任意一条路径找到即 return true（短路），无需继续搜索
- 网格 DFS 边界检查写在函数开头，统一处理越界
