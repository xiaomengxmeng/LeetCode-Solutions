# 📝 2026-05-19 学习记录

## [Medium] Course Schedule II (210)

### 🎯 核心思路
**DFS 三色法 + 后序收集**。与 207 完全相同的有向图环检测逻辑，区别在于检测到环时返回空数组，无环时在 `status[node]=2`（已安全）的时刻把节点加入结果。因为后序先收集子节点，最后需要 reverse。

### 🔑 关键代码片段
```cpp
class Solution {
    vector<int> res;
public:
    vector<int> findOrder(int numCourses, vector<vector<int>>& prerequisites) {
        vector<vector<int>> graph(numCourses);
        for (auto& p : prerequisites)
            graph[p[1]].push_back(p[0]);
        vector<int> status(numCourses, 0);  // 0=未访 1=访问中 2=完成
        for (int i = 0; i < numCourses; i++)
            if (!status[i] && !dfs(graph, status, i))
                return {};                   // 有环
        reverse(res.begin(), res.end());     // 后序收集 → 逆序 → 反转
        return res;
    }
    bool dfs(vector<vector<int>>& graph, vector<int>& status, int node) {
        if (status[node] == 1) return false;
        if (status[node] == 2) return true;
        status[node] = 1;
        for (int neighbor : graph[node])
            if (!dfs(graph, status, neighbor)) return false;
        status[node] = 2;
        res.push_back(node);                 // 后序收集
        return true;
    }
};
```

### 📚 学到的知识点
- **210 = 207 + reverse**：先检测环，无环时后序收集节点再反转
- **后序收集的原理**：DFS 完成后节点才加入，孩子先于父节点 → 结果逆序 → reverse
- **入度法出队顺序天然正确**：不入队不 reverse，更适合输出顺序的场景

### ⚠️ 易错点 & 反思
- ❌ `status` 忘记传引用 → 每层递归拷贝，状态永不更新
- ❌ `return {}` 写成 `return nullptr` → vector 不能返回指针
- ❌ `res.push_back(node)` 写成 `brackets` → 语法错误
- 💡 **三色 DFS + reverse 是输出拓扑顺序的标准模板**
- 💡 **入度法出队即顺序**：更适合输出场景，少一个 reverse

### 🔗 关联题目
- [207] Course Schedule — 只检测环，框架完全相同
- [210] Course Schedule II — 输出拓扑顺序
- [802] Find Eventual Safe States — 三色法找安全节点

## [Medium] Max Area of Island (695)

### 🎯 核心思路
**DFS + 水淹 + 面积累加**。与 [200] 完全相同的网格遍历逻辑，区别在于 DFS 返回当前岛屿的面积：淹掉当前格子后，`1 + 四个方向返回的面积之和`。外层用 `res = max(res, area)` 更新最大值。

### 🔑 关键代码片段
```cpp
class Solution {
public:
    int maxAreaOfIsland(vector<vector<int>>& grid) {
        int dirs[4][2] = {{1,0}, {-1,0}, {0,1}, {0,-1}};
        int m = grid.size(), n = grid[0].size(), res = 0;
        for (int i = 0; i < m; i++)
            for (int j = 0; j < n; j++)
                if (grid[i][j] == 1) {
                    int area = dfs(grid, i, j, dirs);
                    res = max(res, area);
                }
        return res;
    }
    int dfs(vector<vector<int>>& grid, int i, int j, int dirs[4][2]) {
        if (i < 0 || i >= grid.size() || j < 0 || j >= grid[0].size() || grid[i][j] == 0)
            return 0;
        grid[i][j] = 0;
        int sum = 1;
        for (int k = 0; k < 4; k++)
            sum += dfs(grid, i + dirs[k][0], j + dirs[k][1], dirs);
        return sum;
    }
};
```

### 📚 学到的知识点
- **695 = 200 + 面积累加**：框架完全相同，DFS 返回 int 代替 void
- **int 类型 vs char 类型**：`vector<vector<int>>` 的值是 `1` 不是 `'1'`
- **递归返回面积**：`1 + 四方向面积`，天然递归求和

### ⚠️ 易错点 & 反思
- ❌ `int` 类型 grid 用 `== '1'` 比较 → 永远 false，输出 0
- ❌ `vector<vector<int>>` 的 dfs 参数写成 `vector<vector<char>>` → 类型不匹配编译报错
- 💡 **从 200 改到 695 只需改三处**：`void` → `int`，`return` 面积，`max` 取最大值

### 🔗 关联题目
- [200] Number of Islands — 计数版，框架完全相同
- [695] Max Area of Island — 面积版，DFS 返回 int
- [463] Island Perimeter — 计算周长
