# 🧠 算法思路触发框架

> 看到题目 → 识别类型 → 触发思路 → 动手编码

---

## 1️⃣ 双指针

### 🔔 触发信号
- 数组/字符串的**原地操作**（删除、移动、去重）
- 需要**两端向中间**夹逼（回文、两数之和有序版）
- 需要**快慢间隔**（删除倒数第N个、链表找中点）

### 🧩 核心模型
| 变体 | 想到什么 |
|------|---------|
| 同向双指针 | slow走有效位，fast探路 → 移零、去重 |
| 对向双指针 | left右移+right左移 → 两数之和、回文判断 |
| 快慢指针 | fast先走N步 / fast走2步slow走1步 → 倒数删除、链表环 |

### ⚠️ 易忘点
- 链表题先想**虚拟头节点**，省去头节点特殊处理
- 快慢指针找环入口：相遇后，一个从head走，一个从相遇点走，再次相遇即入口

## 2️⃣ 滑动窗口

### 🔔 触发信号
- **连续子数组/子串** + 满足某条件（最长/最短/恰好）
- 关键词：无重复、包含所有字符、和≥target

### 🧩 核心模型
```
初始化：left=0, right=0, 窗口状态(哈希表/计数器)
扩展：right右移，更新窗口状态
收缩：当窗口不满足条件时，left右移，恢复状态
记录：在合法窗口时更新答案
```

| 变体 | 想到什么 |
|------|---------|
| 最长窗口 | 满足条件时扩张，不满足时收缩 |
| 最短窗口 | 满足条件时收缩尝试更短 |
| 固定窗口 | 维护窗口大小=k，滑动取极值 |

### ⚠️ 易忘点
- 窗口收缩时别忘了**恢复状态**（哈希表计数-1，计数器还原）
- 区分"恰好包含"和"至少包含"的收缩时机

## 3️⃣ 哈希表

### 🔔 触发信号
- **O(1)查找**：两数之和、判重、频率统计
- **分组**：字母异位词、按特征归类
- **前缀和优化**：子数组和=目标值 → 前缀和+哈希查差值

### 🧩 核心模型
| 场景 | 想到什么 |
|------|---------|
| 查找配对 | 遍历时查 complement 是否在 map 中 |
| 频率统计 | map[char]++ 或 int[26] 数组 |
| 前缀和+哈希 | sum[i]-target 在 map 中？→ 子数组和=target |
| 树上前缀和 | 路径和+回溯撤销：map[prefix]++ → 递归 → map[prefix]-- |

### ⚠️ 易忘点
- 前缀和要**先放{0:1}**，处理从开头到当前的子数组
- 树上前缀和必须**回溯时撤销**，否则影响其他分支

## 4️⃣ 单调栈

### 🔔 触发信号
- **下一个更大/更小**元素
- **柱状图面积**、**接雨水**
- 需要找"第一个比当前大/小的"

### 🧩 核心模型
```
遍历每个元素：
  while 栈不空 && 当前元素 vs 栈顶（触发条件）：
    弹出栈顶 → 此时当前元素就是栈顶的"答案"
    处理弹出的栈顶（计算面积/距离等）
  当前元素入栈
```

| 变体 | 想到什么 |
|------|---------|
| 下一个更大 | 递减栈（栈顶到栈底递减） |
| 下一个更小 | 递增栈 |
| 循环数组 | 取模遍历 2n 次，或拼接数组 |
| 接雨水 | 按行计算：弹出时 width = i - stack.top() - 1 |

### ⚠️ 易忘点
- 栈存**下标**而非值（需要算距离）
- 循环数组第二遍只弹栈，不重复入栈

## 5️⃣ 二分查找

### 🔔 触发信号
- **有序数组**查找
- **旋转排序数组**
- 求最小/最大的"满足条件的值" → **二分答案**

### 🧩 核心模型

**标准二分**：找目标/插入位置/左边界/右边界
```
left=0, right=n-1
while left<=right:
  mid = left+(right-left)/2
  if 满足条件: 收缩right / 收缩left / 返回
```

**旋转数组**：先判断 mid 在哪半段（有序的那半），再决定搜哪边

**二分答案**：猜一个值 → 验证是否可行 → 调整搜索范围
```
left=最小可能, right=最大可能
while left<right:
  mid = left+(right-left)/2
  if check(mid)可行: right=mid (求最小)
  else: left=mid+1
```

### ⚠️ 易忘点
- lower_bound：`>=target` 的第一个位置
- upper_bound：`>target` 的第一个位置
- 二分答案的 check 函数是关键，先写 check 再套模板

## 6️⃣ 二叉树

### 🔔 触发信号
- 看到 TreeNode / root

### 🧩 核心模型：先想遍历方式！

| 遍历 | 什么时候用 |
|------|-----------|
| 前序（根左右） | 自顶向下传递信息、构造树 |
| 中序（左根右） | BST相关、有序序列 |
| 后序（左右根） | 需要子树结果才能算当前节点、求高度/直径/LCA |
| 层序（BFS） | 按层处理、最短路径、右视图 |

### 🧩 常见套路

| 问题 | 套路 |
|------|------|
| BST验证 | 中序遍历必须递增，或传 (lower, upper) 上下界 |
| 构造树 | 前序定根 → 中序分左右子树 → 递归 |
| LCA | 后序：左找到?右找到? → 当前就是LCA |
| 路径和 | 前缀和+哈希+回溯（同数组前缀和思路） |
| 直径/最长路径 | 后序返回高度，全局变量更新直径 |
| 层序遍历 | queue + 每层size循环 |

### ⚠️ 易忘点
- BST验证用 `long long` 上下界，避开 INT_MIN 边界
- 构造树需要**中序索引的哈希表**加速查找
- 后序求高度：`return 1 + max(left, right)`

## 7️⃣ 图的遍历（DFS/BFS）

### 🔔 触发信号
- **网格/矩阵** + 连通区域/岛屿/面积
- **有向图** + 拓扑排序/环检测
- **最短距离** + 无权图

### 🧩 核心模型

**网格DFS（水淹法）**：
```
dfs(grid, i, j):
  越界或已访问? return
  标记已访问（改值/visited数组）
  四方向递归
```

**有向图环检测（三色DFS）**：
```
0=未访问, 1=访问中, 2=已完成
遇到1 → 有环！
```

**多源BFS**：
```
所有源点同时入队
while queue:
  逐层扩散
  每层 distance++
```

| 问题 | 套路 |
|------|------|
| 岛屿数量 | 网格DFS水淹，计数 |
| 最大岛屿面积 | DFS返回面积，取max |
| 被包围区域 | 边界DFS标记，剩余O翻转 |
| 课程表/拓扑排序 | 三色DFS / BFS入度法 |
| 腐烂橘子/01矩阵 | 多源BFS按层计时 |

### ⚠️ 易忘点
- 网格DFS方向数组：`{0,1,0,-1,0}`
- 多源BFS：**所有源点先入队**，再统一BFS
- 边界DFS题：先从四边标记，再处理内部

## 8️⃣ 回溯

### 🔔 触发信号
- **所有可能的组合/排列/子集**
- **网格搜索路径**
- 关键词：枚举、所有方案、是否存在路径

### 🧩 核心模型
```
backtrack(路径, 选择列表):
  if 满足结束条件:
    收集结果
    return
  for 选择 in 选择列表:
    做选择（路径添加 + 标记used）
    backtrack(路径, 选择列表)
    撤销选择（路径移除 + 取消标记）
```

| 变体 | 选择列表 | 去重方式 |
|------|---------|---------|
| 全排列 | 所有未选元素 | used数组 |
| 子集 | 当前位置之后 | 起点start |
| 组合总和 | 当前位置之后（可重复则含当前） | 排序+跳过相同 |
| 网格搜索 | 四方向 | 原地标记+回溯恢复 |
| 括号生成 | 左/右括号 | 左<n可加左，右<左可加右 |

### ⚠️ 易忘点
- 组合题排序后可以**剪枝**：剩余和 < 0 提前返回
- 子集 vs 排列：子集用 start 避免重复顺序，排列用 used 避免重复选
- 网格回溯：标记 → 递归 → **恢复标记**

## 🔀 混合题型识别

| 看到这些 | 可能是 |
|---------|--------|
| 连续子数组 + 和/条件 | 滑动窗口 or 前缀和+哈希 |
| 下一个更大 + 循环 | 单调栈 |
| 有序 + 查找/最值 | 二分 |
| 有序 + 旋转 | 旋转二分 |
| 树 + 最值/路径 | 后序+全局变量 |
| 树 + BST | 中序 or 上下界 |
| 网格 + 连通 | DFS水淹 |
| 网格 + 最短距离 | 多源BFS |
| 有向图 + 顺序/环 | 拓扑排序 |
| 枚举所有方案 | 回溯 |
| 链表 + 倒数/环 | 快慢指针 |
| 链表 + 反转/删除 | 虚拟头节点 |
