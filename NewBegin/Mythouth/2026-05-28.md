## [Medium] Permutations (46)
### 🤔 我的原始思路
[在此记录你的初步想法]
第一个位置有三种选择 选定第一个后 第二个位置有两种选择 第三个位置有一个选择
vector<bool> used  递归中 使用数字 标记为 true  回到上一层不使用数字 标记回false
递归终止条件：当路径长度等于 nums长度时，将路径添加到结果中
### 🔍 我的思考过程
- 回溯三件套：标记 used → 递归 → 撤销 used
- 终止条件：path.size() == nums.size()
- used 需要初始化大小

### 💡 对比收获
- **回溯 = DFS + 状态重置**：沿着决策树走下去，走不通或到达终点就回溯
- **相比图 DFS 的区别**：图 DFS 标记后不需要撤销；回溯的"状态"（used）需要在递归返回后撤销
