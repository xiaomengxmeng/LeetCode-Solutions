## [Medium] Find First and Last Position of Element in Sorted Array (34)
### 🤔 我的原始思路
一次找第一个 >= target 的位置，一次找最后一个 <= target 的位置
可以 说明 不存在目标值
### 🔍 我的思考过程
- lower_bound 返回第一个 >= target 的位置，nums[mid] >= target 时收缩 right
- 右边界 = lower_bound(target+1) - 1
- leftbound == rightbound 说明不存在
### 💡 对比收获
- **35 → 34 的升级**：从"找插入点"到"找左右边界"，lower_bound 是关键工具
- **核心变化**：`if (nums[mid] >= target)` 而不是 `if (nums[mid] > target)`，等号也收缩才能找到最左边
- **配套使用**：lower_bound(target) 和 lower_bound(target+1) 一前一后确定区间
